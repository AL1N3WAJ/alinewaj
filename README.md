## ⚙️ Deployment (GitHub Only)

This project uses GitHub Actions for automatic deployment:

1. Upload all files to your repository
2. Go to:

Settings → Pages
3. Set:

Source → GitHub Actions
4. Done ✅

Site will deploy automatically on every push.
